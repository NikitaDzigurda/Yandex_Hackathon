# AI-платформа автоматизации управления проектной деятельностью

Решение кейса "Платформа автоматизации управления проектной деятельностью на базе ИИ-агентов".
Система использует мультиагентную архитектуру (Multi-Agent System) для оценки, глубокого ресёрча и мониторинга проектных заявок.

## 🌟 Ключевые особенности решения
- **Проверено на исторических данных:** Система провалидирована на реальном датасете Центра (HISTOSCAN, DermaMarker и др.) и показала точное совпадение решений ИИ-агента с решениями экспертов.
- **Human-in-the-Loop (HITL):** ИИ не принимает финальных решений о выделении ресурсов. Система ставит процессы на паузу до получения апрува от Руководителя Проекта (РП).
- **Событийно-ориентированная архитектура:** Интеграция агентов через брокер сообщений (Redis), что обеспечивает отказоустойчивость и возможность горизонтального масштабирования.

## 🏗 Архитектура системы
1. **Intake Agent (YandexGPT):** Принимает заявку, оценивает по 5 критериям, проверяет релевантность облачных технологий.
2. **Research Agent (YandexGPT):** Проводит глубокий доменный анализ одобренных проектов, генерирует гипотезы и выявляет риски.
3. **Monitor Agent:** Отслеживает статусы задач в Yandex Tracker и активность в репозиториях Sourcecraft.
4. **Orchestrator:** Координирует асинхронные задачи агентов через Redis.

*Примечание к архитектуре:* В текущем MVP Orchestrator работает как фоновая задача FastAPI. В production-среде он выносится в отдельный микросервис (на базе Celery или Arq) для независимого скейлинга.

## 🚀 Быстрый запуск (DEPLOY)
Смотрите инструкции в файле `DEPLOY.md`.
Система разворачивается одной командой `docker compose up --build`.

## 🎬 Как тестировать (Сценарий демо)
1. Откройте UI: `http://localhost:8000`
2. В Секции 6 нажмите **"Загрузить демо заявки"**. В базу загрузятся реальные кейсы (2 ИТ/Медицина, 2 из других сфер без ИТ-составляющей).
3. Запустите Intake Agent. Обратите внимание в Секции 7 ("Сводка проекта"), как агент *обоснованно отклоняет* не-ИТ проекты и *рекомендует к одобрению* профильные проекты.
4. Одобрите целевую заявку через Секцию 5 (Решение РП).
5. Запустите Research Agent и получите глубокий структурированный отчёт.
