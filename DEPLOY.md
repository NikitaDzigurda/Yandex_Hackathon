# DEPLOY.md

Этот файл предназначен **только** для агента-оценщика: здесь должны быть воспроизводимые команды, которые позволяют развернуть и протестировать решение на чистом окружении.

## Предусловия

- Docker установлен и запущен
- Docker Compose доступен как `docker compose`

## Быстрый запуск

```bash
docker compose up -d --build
```

## Проверка работоспособности

Опишите, как проверить, что система поднялась корректно:

- URL/порт(ы)
- ожидаемый ответ/страница/лог
- тестовые учетные данные (если нужны)

## Тесты / проверки

```bash
# пример — замените на ваши команды
docker compose exec <service> <command>
```

## Остановка и очистка

```bash
docker compose down -v
```
